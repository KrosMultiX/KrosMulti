## ✨ Fonctionnalités



### 👥 Gestion de Team
- Créez et sauvegardez plusieurs compositions de team
- Ajoutez / retirez des personnages en un clic
- Team de base + teams secondaires illimitées
- Slots visuels avec avatars de personnages

### 🧑 Vue par Personnage
- Liste compacte de tous vos comptes actifs
- Accès rapide à chaque personnage
- Statut en temps réel avec avatar
- Navigation par onglets entre les comptes

### 🛒 Trading Intégré
- Gérez les échanges entre vos comptes sans quitter le jeu
- Interface dédiée par personnage

### ⚡ Auto Focus
- Changement de fenêtre automatique entre vos comptes
- Synchronisation des actions multicompte

### 🎛 Mode Confort
- Suppression des pop-ups et notifications in-game
- Affichage du **mode tactique** permanent
- **SpeedHack** configurable
- Désactivation des animations pour optimiser les performances

### 🔒 Multicompte HWID
- Changement de HWID (spoofing) pour jouer sur plusieurs comptes sur les **serveurs monocomptes**
- Sans recours à des machines virtuelles
- Compatible AvA, percepteurs en équipe

### 🔧 Performances
- Réduction légère de la consommation **RAM** et **CPU**
- Interface overlay légère non intrusive
- Compatible avec toutes les résolutions

## 📦 Installation

### Prérequis
- Windows 10 / 11
- Dofus Unity ou Dofus Retro installé

### Téléchargement

```bash
# Option 1 — Téléchargement direct
https://www.krosmulti.com/#download

# Option 2 — Cloner le dépôt
git clone https://github.com/votre-username/krosmulti.git
cd krosmulti
```

### Lancement
1. Téléchargez `Krosmulti.exe` depuis [krosmulti.com](https://www.krosmulti.com)
2. Lancez Dofus (Unity ou Retro)
3. Démarrez Krosmulti — l'overlay se superpose automatiquement
4. Configurez vos comptes dans l'onglet **Personnages**
